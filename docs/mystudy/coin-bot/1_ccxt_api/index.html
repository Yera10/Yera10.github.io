      1. API Key 발급 #

      Upbit 홈페이지 > 우측 하단 Open API > Open API 사용하기
      위 경로로 들어가 권한을 선택하고 IP주소를 등록하면 Key를 발급받을 수 있다.
      권한은 혹시 해킹당할 수 있을까봐 출금하기, 입금하기만 빼고 모든 권한을 선택했다. Access Key와 Secret Key를 upbit.key라는 이름으로 저장해 놓는다.

      2. 거래소 객체 생성 #

      import ccxt
import pprint
import pandas as pd

# API Key 불러오기
with open("./upbit.key") as f:
    lines = f.readlines()
    api_key = lines[0].strip()
    api_secret = lines[1].strip()

# 거래소 객체 생성
exchange = ccxt.upbit(config=dict(
    apiKey = api_key, 
    secret = api_secret, 
    enableRateLiit = True
))

print(exchange)

      API Key는 잔고조회, 매수, 매도를 사용할 때만 필요했고 현재가 조회나 티커 조회를 할 때는 없어도 됐었다.

      6. 시세 캔들 조회 #

      # 시세 캔들 조회

# 분봉
ohlcv = exchange.fetch_ohlcv(symbol="BTC/KRW", timeframe='1m')

pd_ts = pd_ts.dt.tz_convert("Asia/Seoul")                   # convert timezone
pd_ts = pd_ts.dt.tz_localize(None)
df.set_index(pd_ts, inplace=True)
