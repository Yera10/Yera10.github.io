      python2에서는 써드파티 라이브러리인 virtualenv 가상환경 라이브러리를 사용
      python3에서는 기본적으로 제공되는 venv 가상환경 라이브러리를 사용

      가상환경 생성 #

      {가상환경이름}\Scripts\activate.bat

source {가상환경이름}\Scripts\activate.bat
